# C2 Race Cars Tire Tracker

A portrait-oriented Progressive Web App for tracking:

- Left-front, right-front, left-rear, and right-rear tire PSI
- Tire circumference in inches
- Front stagger: RF minus LF
- Rear stagger: RR minus LR

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload every file and folder from this package while preserving the folder structure.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`.
6. Save.

Your app will be available at:

`https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/`

## Install on an iPhone Home Screen

1. Open the GitHub Pages address in Safari.
2. Tap the Share button.
3. Tap **Add to Home Screen**.
4. Tap **Add**.

The app includes an Apple touch icon, a web manifest, and offline caching.
